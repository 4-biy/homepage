Anda Berada di Luar Area yang Ditentukan

Mohon pastikan perangkat Anda berada dalam radius yang benar untuk melakukan absensi. Jika Anda masih mengalami kendala, ikuti beberapa langkah berikut untuk memperbaiki akurasi lokasi di perangkat Anda.

Langkah-Langkah untuk Mengatasi Lokasi Tidak Akurat:

  1. Restart Layanan Lokasi: Matikan layanan lokasi (GPS) di pengaturan ponsel Anda, lalu nyalakan kembali.
  2. Pindahkan Posisi Anda: Berjalanlah sedikit di sekitar area terbuka untuk membantu GPS mendapatkan sinyal lebih akurat.
  3. Aktifkan Mode Akurasi Tinggi: Buka pengaturan lokasi di ponsel Anda, pilih mode "Akurasi Tinggi" agar perangkat dapat menggunakan Wi-Fi, Bluetooth, dan jaringan seluler.
  4. Pastikan Aplikasi di Izin Lokasi: Pastikan aplikasi ini memiliki izin akses lokasi. Buka "Pengaturan > Aplikasi > [Nama Aplikasi Browser] > Izin" dan pastikan izin lokasi diaktifkan.
  5. Hapus Data Aplikasi: Anda bisa mencoba menghapus data aplikasi. Buka "Pengaturan > Aplikasi > [Nama Aplikasi Browser] > Penyimpanan", lalu pilih "Hapus Data" atau "Hapus Cache". Pastikan untuk membuka kembali aplikasi setelahnya.
  6. Hapus Data di Browser: Jika masalah tetap berlanjut, Anda bisa mencoba menghapus data browser seperti cache dan cookies. Ikuti langkah berikut:
    1. Buka pengaturan browser Anda.
    2. Pilih "Privasi" atau "Pengaturan Data" (tergantung browser yang Anda gunakan).
    3. Pilih "Hapus Data Penjelajahan" atau "Hapus Cache dan Cookies".
    4. Pilih jenis data yang ingin Anda hapus (cache, cookies, dan data lainnya), lalu tekan "Hapus".
    Setelah menghapus data, buka kembali halaman absensi dan coba lagi.

Langkah untuk Mengaktifkan Lokasi di Perangkat Anda:

Untuk **Android**:

  1. Langkah 1: Buka Pengaturan di ponsel Anda.
  2. Langkah 2: Pilih Lokasi.
  3. Langkah 3: Pastikan Lokasi diaktifkan.

Untuk **iOS**:

  1. Langkah 1: Buka Pengaturan di iPhone Anda.
  2. Langkah 2: Pilih Privasi dan kemudian pilih Layanan Lokasi.
  3. Langkah 3: Aktifkan Layanan Lokasi.
